## Available Trigger Events

### New Member Offer Access

**Event Name:** `member.access.granted`
**Description:** Triggered when a new or existing member gets access to an offer.
**Payload Example:**

```json
{
  "event": "member.access.granted",
  "member_id": "mem_123",
  "product_id": "prd_456",
  "product_type": "course",
  "access_granted_at": "2024-01-20T15:30:00Z",
  "access_level": "full"
}
```

### New Payment

**Event Name:** `payment.succeeded`
**Description:** Triggered when a successful payment is processed an offer.
**Payload Example:**

```json
{
  "event": "payment.succeeded",
  "payment_id": "pay_123",
  "member_id": "mem_456",
  "amount": 99.00,
  "currency": "USD",
  "product_id": "prd_789",
  "payment_method": "card",
  "timestamp": "2024-01-20T15:30:00Z"
}
```
