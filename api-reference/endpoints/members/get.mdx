openapi: get /members

## Response Parameters

| Field       | Type   | Description                              |
| ----------- | ------ | ---------------------------------------- |
| id          | string | Unique identifier for the member         |
| email       | string | Member's email address                   |
| first\_name | string | Member's first name                      |
| last\_name  | string | Member's last name                       |
| created\_at | string | ISO 8601 timestamp of member creation    |
| updated\_at | string | ISO 8601 timestamp of last member update |
